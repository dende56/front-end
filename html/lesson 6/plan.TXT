3. Підключення стилів 
   3.1.  Поки підключаємо стилі у html через тег <style> </style>
            *Якщо хочете підключити стилі через файл .css то у папці assets створюємо папку сss і у ній файл style.css. 
            Відносний шлях від нашого файлу html має бути ./assets/css/style.css. Стилі написані у тегу <style></style> мають більший пріоритет над style.css

4. Основні властивості css для елементів:
      4.1.  Колір фону (про те, як саме задається колір і в яких одиницях поговоримо на уроці) :
                  background-color: #000;
                  background-color: #cfcfcf;
                  background-color: red;
                  background-color: green;

      4.2.  Рамка (про те, які види рамок існують, можете загуглити, також про це поговоримо на уроці, ми використовуємо суцільну лінію - solid):
            border: [ширина лінії] [вид лінії] [колір лінії];
            border: 2px solid #000;
            border: 10px solid red;
            border: 5px solid rgb(0, 255, 34);
            
            *Можна задати заокруглення рамки (про те, які можливі аргументи поговоримо на уроці), щоб помітити зміни обов'язково має бути рамка 
            border: 2px solid #000;
            border-radius: 6px;

            border: 2px solid #red;
            border-radius: 20px;

      4.3   Вирівнювання тексту (відтепер ми небудемо використовувати атрибут align): 
            text-align: center;
            text-align: start;
            text-align: end;

      4.4   Розмір тексту (про одиниці у яких задається значення поговоримо на уроці, поки використовуємо пікселі ( px )):
            font-size: 20px;
            font-size: 16px; (Таке значення використовується за змовчуванням для звияайного тексу у параграфі ( <p> </p> )
            font-size: 30px;
            *font-size: 2em;
